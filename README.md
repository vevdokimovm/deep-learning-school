# deep-learning-school

Course materials from the **Deep Learning School** by МФТИ / ФПМИ (2018–2019).  
Seminars, homeworks, and lecture notebooks covering the fundamentals of deep learning.

> Originally forked from the official course repository.

## Topics

1. Introduction to AI & DL tools — Python basics, Google Colab, NumPy
2. Linear Algebra & NumPy
3. Linear Models & Logistic Regression
4. Neural Networks basics
5. Convolutional Neural Networks (CNN)
6. Recurrent Neural Networks (RNN)
7. ... and more

## Tech Stack

- Python 3
- Jupyter Notebook
- NumPy, Pandas, Matplotlib
- PyTorch / TensorFlow
